# 年份倒计时
- 展示当前年份的倒计时(下一年元旦，即下一年1月1日)
- 动态更新时间，无需按烂Ctrl+R
- 简介的用户界面，只有时间与背景

在一年结束后，自动切换下一年倒计时。

一些功能还在制作中，欢迎提出建议。
